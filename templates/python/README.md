# Python template

This is a minimal ament_python package template with shared helpers.
- Rename the package folder and references in `package.xml`, `setup.py`, and `setup.cfg`.
- Update the node name in `lesson_py_template/node.py`.
- Common helpers live in `lesson_py_template/qos.py`, `lesson_py_template/topics.py`,
  and `lesson_py_template/services.py`.
