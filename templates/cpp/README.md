# C++ template

This is a minimal ament_cmake package template with header/source separation.
- Rename the package in `package.xml` and `CMakeLists.txt`.
- Update the node name in `include/lesson_cpp_template/lesson_node.hpp`.
- The `main` entry point lives in `src/lesson_node.cpp`.
- Common helpers live in `include/lesson_cpp_template/qos.hpp`,
  `include/lesson_cpp_template/topics.hpp`, and
  `include/lesson_cpp_template/services.hpp`.
